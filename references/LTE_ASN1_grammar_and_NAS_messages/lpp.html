LTE Positioning Protocol (LPP)

36.355 v12.2.0 (2014-06)

Protocol between a location server (E-SMLC or SLP) and a target device (UE or SET)

  • Obtain location related measurements (OTDOA, E-CID) or a location estimate (A-GNSS)
  • Request / Provide :
    • User equipment (UE) positioning capabilities
    • Assistance Data
    • Location Information
